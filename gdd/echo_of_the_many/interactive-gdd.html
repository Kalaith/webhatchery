🎮 Game Overview

▼
Echo of the Many is a fantasy strategy game where players control a mage who creates magical clones to infiltrate and influence a medieval city's political landscape.

🧙‍♂️ Clone Management

Create and manage multiple magical clones, each with unique disguises and abilities. Deploy them across different city districts to gather intelligence and influence key figures.

🏛️ Faction Politics

Navigate complex relationships between Nobles, Merchants, Crafters, and Scholars. Your actions affect faction influence and unlock new opportunities.

🗺️ District Exploration

Explore Noble Quarter, Market District, Artisan Ward, Scholar's Enclave, and Slums. Each area offers unique missions and challenges.

📱 Mobile-First Design

Optimized for touch interactions with responsive design, offline play capabilities, and Progressive Web App features.

🎯 Core Game Pillars

  • Strategic Deception: Use clones to infiltrate without detection
  • Political Intrigue: Manipulate faction relationships for personal gain
  • Resource Management: Balance energy, influence, and risk
  • Progressive Discovery: Unlock new areas and abilities through success

⚔️ Core Gameplay Mechanics

▼

🎭 Clone System

Clone Creation

Spend energy to create clones with specific disguises:

  • Noble: High influence, expensive to maintain
  • Merchant: Good for trade and information
  • Artisan: Access to crafting areas
  • Scholar: Academic knowledge and research
  • Beggar: Low cost, high mobility

Mission Assignment

Assign clones to various activities:

  • Gather Intelligence
  • Influence Key Figures
  • Sabotage Operations
  • Build Relationships
  • Acquire Resources

🏰 Faction Influence System

Nobles (30% Initial Influence)

Control political decisions and military. High influence unlocks administrative positions.

Merchants (25% Initial Influence)

Control trade and economy. Good relationships provide financial resources.

Crafters (20% Initial Influence)

Provide services and goods. Access to tools and magical components.

Scholars (15% Initial Influence)

Control knowledge and magic. Essential for learning new spells.

🪄 Magic System

Clone Spells

  • Mirror's Edge: Create basic clone (10 energy)
  • Second Skin: Advanced disguise (15 energy)
  • Whispers of Self: Information gathering (8 energy)

Utility Spells

  • Veil of Shadows: Reduce detection risk
  • Golden Tongue: Increase persuasion
  • Mind's Eye: Reveal hidden information
Risk Management: Detected clones can damage faction relationships and expose your operations. Plan carefully!

🔧 Initial Setup Commands

# Create new Vite React TypeScript project
npm create vite@latest echo-of-the-many -- --template react-ts

# Navigate to project directory
cd echo-of-the-many

# Install dependencies
npm install

# Install additional packages
npm install zustand react-spring @use-gesture/react
npm install -D @types/node tailwindcss autoprefixer postcss

# Development dependencies
npm install -D eslint @typescript-eslint/eslint-plugin
npm install -D prettier eslint-config-prettier
npm install -D @vitejs/plugin-pwa workbox-window

# Start development server
npm run dev
